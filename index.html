Установка Gulp в свой проект

Привет, друг! Пишу этот мануал для себя любимого, чтобы в будущем не тратить время на установку Gulp в свой проект. Ниже подробно опишу каждый из этапов установки. Думаю, что мне будущему это понравится ;)

  • Установка Node.js
  • Глобальная установка Gulp
  • Инициализация
  • Установка пакетов в проект
  • gulpfile.js

1. Установка Node.js

Для работы с npm-командами необходимо загрузить Node.js на свой компьютер. Загрузка с официальнольного сайта Node.js.
Все последующие действия выполняем в терминале из-под папки с нашим проектом!

2. Глобальная установка Gulp

После установки Node.js нам необходимо установить Gulp на наш компьютер. Сперва открываем терминал (cmd.exe) от имени администратора и вводим команду:

npm install -g gulp

Отлично, теперь Gulp есть на нашем компьютере. Приступаем к следующему шагу.

3. Инициализация

Теперь нам необходимо прописать следующую команду:

npm init

После чего терминал нас попросит некоторые данные о проекте, такие как автор, ключевые слова и прочее. Это можно пропуск, нажимая ENTER. Если всё было сделано правильно, то в папке с проектом появится файл «package.json».

4. Установка пакетов в проект

Чтобы всё работало корректно, необходимо прописать следующие команды:

Gulp собственной персоной

npm i --save-dev gulp

BrowserSync, который позволит нам автоматически обновлять страницу при каждом изменении в файлах

npm i --save-dev browser-sync

Рекомпилятор из .sass в .css

npm i --save-dev gulp-sass

Автопрефиксер

npm i --save-dev gulp-autoprefixer

Конкатенация sass-файлов в единый css-файл

npm i --save-dev gulp-concat-css

Выгрузка файлов на хостинг. Подробнее его работу и API можно посмотреть здесь.

npm i --save-dev gulp-ftp

Отлично, теперь можно переходить в следующему пункту

5. gulpfile.js

gulpfile.js - главный винец торжества. В нём прописан весь код для работы установленных компонентов. Скачать можно здесь.
Его необходимо закинуть в папку с проектом, после чего можно запускать Gulp следующей командой:

gulp

Теперь можно работать. Наш маленький раб будет делать за нас всю работу, а конкретно: автообновление страницы, рекомпиляция и конкатенация .sass-файлов в .css-файл, автоматическая расстановка префиксов и, по желанию, выгрузка на наш хостинг.
